# Telegram-бот кофейни «Щегол»

Бот работает отдельно от сайта: показывает меню, помогает собрать предзаказ, открывает филиалы, отзывы, новинки недели и карту гостя.

## 1. Создать бота в BotFather

1. Откройте Telegram.
2. Найдите `@BotFather`.
3. Отправьте команду `/newbot`.
4. Задайте название и username бота.
5. BotFather выдаст токен вида `123456:ABC...`.

## 2. Добавить токен

Скопируйте `.env.example` в `.env`.

Для Windows PowerShell:

```powershell
copy .env.example .env
```

Для macOS/Linux:

```bash
cp .env.example .env
```

В файле `.env` замените значение:

```env
BOT_TOKEN=ваш_токен_от_BotFather
```

## 3. Установить зависимости

Проект написан без привязки к версии Python и проверен по синтаксису на Python 3.14.6. Если `pip` при установке `aiogram` сообщит о несовместимости, поставьте Python 3.12 как запасной стабильный вариант.

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## 4. Запустить бота

```bash
python bot.py
```

На Windows можно просто открыть файл:

```text
run_bot.bat
```

После запуска откройте своего бота в Telegram и отправьте `/start`.

## 5. Что можно менять

- `data.py` - филиалы, ссылки на карты и отзывы, новинки недели, варианты молока, сахара и добавок.
- `../assets/menu-items-full.json` - основное меню сайта и бота.
- `assets/yandex-photo-urls.csv` - публичные ссылки на фотографии для карточек меню.
- `keyboards.py` - кнопки и структура клавиатур.
- `bot.py` - логика корзины, предзаказа, карты гостя и сообщений.

Меню загружается из файла сайта `assets/menu-items-full.json`. В боте одинаковые позиции с разными объёмами объединяются в одну карточку, поэтому кофе сейчас отображается как 9 позиций: `Фильтр 0.2/0.3` объединён в `Фильтр`, `Капучино 0.2/0.3` объединён в `Капучино`.
Фотографии в Telegram берутся только из публичных URL. Локальные пути к файлам не используются для карточек товаров.

## Ссылка для сайта

Когда у бота будет username, на сайте можно поставить ссылку:

```text
https://t.me/USERNAME_BOT
```

Например, кнопки: «Заказать в Telegram», «Открыть меню в Telegram», «Карта гостя».

## Что ещё нужно для реального запуска

Сейчас корзина и карта гостя хранятся в памяти. После перезапуска прогресс пропадает. Для боевого запуска лучше подключить SQLite и уведомления бариста о новых заказах.
